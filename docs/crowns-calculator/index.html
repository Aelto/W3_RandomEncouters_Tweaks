Crowns calculator for Random Encounters Reworked

Quick explanation

The amount of crowns dropped by RER creatures is calculated dynamically based on the creatures size. It uses a logarithmic function and a few other variables such as the base amount, the size scaling and the scaling offset to control how many crowns are dropped.

Below is a dynamic form where you can enter different values for the variables and see the result for key creatures like the ghouls, werewolves and fiends.
