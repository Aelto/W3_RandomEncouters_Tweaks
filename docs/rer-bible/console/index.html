Console commands

RER has a few console commands you may want to use. Here is a list of them and how to use them

rerkillall will kill every creatures spawned by RER in the world. Creatures killed this way do not drop anything.

rerchallenge(seed) where seed is a number creates a simplified bounty in the world. You can also use the seed 0 to generate a completely random bounty. Learn more about bounties in the bounties page

rera(creature-type) where creature-type can be one of the following thing:

  • Small creatures below
  • CreatureHUMAN
  • CreatureARACHAS
  • CreatureENDREGA
  • CreatureGHOUL
  • CreatureALGHOUL
  • CreatureNEKKER
  • CreatureDROWNER
  • CreatureROTFIEND
  • CreatureWOLF
  • CreatureWRAITH
  • CreatureHARPY
  • CreatureSPIDER
  • CreatureCENTIPEDE
  • CreatureDROWNERDLC
  • CreatureBOAR
  • CreatureBEAR
  • CreaturePANTHER
  • CreatureSKELETON
  • CreatureECHINOPS
  • CreatureKIKIMORE
  • CreatureBARGHEST
  • CreatureSKELWOLF
  • CreatureSKELBEAR
  • CreatureWILDHUNT
  • CreatureBERSERKER
  • CreatureSIREN
  • Large creatures below
  • CreatureDRACOLIZARD
  • CreatureGARGOYLE
  • CreatureLESHEN
  • CreatureWEREWOLF
  • CreatureFIEND
  • CreatureEKIMMARA
  • CreatureKATAKAN
  • CreatureGOLEM
  • CreatureELEMENTAL
  • CreatureNIGHTWRAITH
  • CreatureNOONWRAITH
  • CreatureCHORT
  • CreatureCYCLOP
  • CreatureTROLL
  • CreatureHAG
  • CreatureFOGLET
  • CreatureBRUXA
  • CreatureFLEDER
  • CreatureGARKAIN
  • CreatureDETLAFF
  • CreatureGIANT
  • CreatureSHARLEY
  • CreatureWIGHT
  • CreatureGRYPHON
  • CreatureCOCKATRICE
  • CreatureBASILISK
  • CreatureWYVERN
  • CreatureFORKTAIL
  • CreatureSKELTROLL
Or CreatureNONE for a random creature. This command creates an ambush after a five seconds delay.

rerh(creature-type) where creature-type can be any value from the list above. The command will create a hunt after a five seconds delay.

rerhg(creature-type) where creature-type can be any value from the list above. The command will create a hunting ground after a five seconds delay.

rerhorde(creature-type) where creature-type can be any value from the list above. The command will create a horde around you. You can stack hordes and have them running at the same time.

rerhu(human-type, count) where count is a number and human-type can be any value from this list:

  • HT_BANDIT
  • HT_NOVBANDIT
  • HT_SKELBANDIT
  • HT_SKELBANDIT2
  • HT_CANNIBAL
  • HT_RENEGADE
  • HT_PIRATE
  • HT_SKELPIRATE
  • HT_WITCHHUNTER
The command will create a human ambush after a five seconds delay.

rerremoveallpins, with the arrival of the bounties you may have pins in weird positions and you can't remove them. This command is useful for that, it removes pins from all maps no matter their position and type.

rerabandonbounty, to cancel the currently active bounty.
